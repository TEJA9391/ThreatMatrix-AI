# 🛡️ ThreatMatrix AI | Advanced SOC Mainnet

**ThreatMatrix AI** is a production-grade Security Operations Center (SOC) platform designed for real-time threat intelligence, neural visualization, and autonomous mitigation.

![ThreatMatrix Dashboard](https://images.unsplash.com/photo-1550751827-4bd374c3f58b?w=1200)

## 🚀 Key Features
- **Neural Intelligence Mesh**: High-fidelity visualization of global threat ingress and core synaptic processing.
- **Autonomous Mitigation**: Real-time auto-blocking of malicious IPs with live incident logging.
- **Multi-Vector Detection**: Dedicated neural engines for Fraud, Phishing, and Fake News identification.
- **Premium UX**: Immersive "Cyberpunk" aesthetic with glassmorphism and fluid Framer Motion transitions.

## 🛠️ Technology Stack
- **Frontend**: React 18, Tailwind CSS, Lucide Icons, Recharts, Framer Motion.
- **Backend**: Python Flask, Socket.IO (Real-time Telemetry).
- **Architecture**: Zero-Trust design with neural-synced telemetry buffers.

## 📦 Installation & Setup

### Prerequisites
- Node.js (v18+)
- Python 3.9+
- Pip

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
