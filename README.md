# ThreatMatrix AI: Production-Grade SOC Platform

ThreatMatrix AI is a production-grade, high-fidelity **Security Operations Center (SOC)** platform designed to detect, visualize, and neutralize global cyber threats in real-time.

## 🚀 Project Overview
This project features a real-time **Neural Intelligence Mesh**, an **Autonomous Mitigation** system, and specialized detection modules for Fraud, Phishing, and Fake News.

## 🛠️ Project Structure
- **/frontend**: React 18 + Vite + Tailwind CSS + Framer Motion.
- **/backend**: Python Flask + Socket.IO Neural Kernel.

## ⚙️ Quick Start

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 🎨 Architecture
- **Neural Matrix**: Multilayered visualization (Cyan: Edge, Purple: Core).
- **Socket Mainnet**: High-speed real-time data synchronization.
- **Zero-Trust UI**: Professional glassmorphic command center.
